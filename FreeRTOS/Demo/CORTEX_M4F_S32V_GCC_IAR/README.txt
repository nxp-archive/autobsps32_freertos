This package contains a demo application for FreeRTOS V8.2.3 running on s32v234.

The location of the demo application relative to the FreeRTOS root directory is
FreeRTOS\Demo\CORTEX_M4F_S32V_GCC_IAR

The sample application can be executed on the S32V234-EVB.

To build the application use the makefile in the current folder.
To execute the application use the START.bat file generated by the build process.

The application will toggle pin 17 on J24 (Expansion Connector) every 200ms.

s32v234_board.jpg shows the board and the pin toggling.
s32v234_running_T32.png shows the T32 debugger window with the demo running.

The following environment variables must be set in order to build this application.

LAUTERBACH_PATH       - path to the Lautherbach Trace32 debugger
GCCARM_TOOLCHAIN_PATH - path to the GCC toolchain
IARARM_TOOLCHAIN_PATH - path to the IAR toolchain

example:
export LAUTERBACH_PATH=c:/tools/T32
export GCCARM_TOOLCHAIN_PATH=c:/tools/gcc-linaro-4.9-2014.11
export IARARM_TOOLCHAIN_PATH=c:/tools/IAR_7403_8938

This release was validated with the following compilers:

GCC : (GNU Tools for ARM Embedded Processors) 4.9.3 20150529 (release) [ARM/embedded-4_9-branch revision 224288] 
IAR : IAR ANSI C/C++ Compiler V7.40.3.8902/W32 for ARM

in order to build the demo application use "make" in the current folder

example:

# build using GCC compiler
make compiler=gcc 

# build using IAR compiler
make compiler=iar 

# clean-up
make clean 


The directory structure is as follows:

./:
    README.txt                                        ; this file
    FreeRTOSConfig.h                                  ; FreeRTOS Configuration for this demo application
    main.c                                            ; demo main code
    makefile                                          ; demo makefile
    s32v234.cmm                                       ; T32 CMM script
    s32v234_board.png                                 ; the S32V234-EVB running this demo
    s32v234_running_T32.png                           ; screenshot with T32 running this demo

./Freescale_Code/common:
    assert.h                                          ; macro for software assertions
    common.h                                          ; File to be included by all project files
    startup.c                                         ; Generic s32v234 startup code

./Freescale_Code/compilers/gcc:
    crt0.s                                            ; statup code for GCC compiler
    gcc.h                                             ; GCC compiler specific include file

./Freescale_Code/compilers/gcc/linker:
    gcc_ram.ld                                        ; GCC liker script

./Freescale_Code/compilers/iar:
    crt0.s                                            ; statup code for IAR compiler
    iar.h                                             ; IAR compiler specific include file

./Freescale_Code/compilers/iar/linker:            
    iar_ram.icf                                       ; IAR liker script

./Freescale_Code/cpu:
    arm_cm4.c                                         ; Generic high-level routines for ARM Cortex M4 processors
    arm_cm4.h                                         ; Definitions common to all ARM Cortex M4 processors
    start.c                                           ; C code entry point
    sysinit.c                                         ; s32v234 Configuration
    sysinit.h                                         ; s32v234 Configuration
    vectors.c                                         ; M4 interrupt vector table
    vectors.h                                         ; M4 interrupt vector table

./Freescale_Code/platform:
    S32V234.h                                         ; register and bit field definitions for the S32V234
